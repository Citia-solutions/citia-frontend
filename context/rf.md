RF-00: Inicio de sesión de usuario:El sistema debe permitir a los usuarios
iniciar sesión utilizando un nombre de usuario y contraseña válidos.
RF-01: Monitoreo: El sistema debe permitir a los usuarios gestionar y   visualizar el comportamiento y seguimiento del paciente.
RF-02: Calendario: EL sistema debe permitir sincronizar con los calendarios que maneje el usuario
RF-03: Dashboard de citas del día: El sistema debe permitir tener un layout con las citas del día incluyendo datos del paciente, hora y fecha.
RF-04: Perfil: El sistema debe permitir tener un vista para las configuraciones del perfil del usuario incluyendo suscripción, soporte etc.
RF-05: Alertas: El sistema debe enviar alertas al usuario en caso de los siguiente eventos: reagendamiento, cancelación de cita, solicitud de cita.
RF-06 — Recordatorios al paciente: programar y enviar recordatorios automáticos al paciente (canal a definir: WhatsApp/SMS/email) en momentos configurables. (Aquí cuelga tu RNF-03.)
RF-07 — Respuesta del paciente: el paciente confirma, cancela o solicita reagendar desde el recordatorio; el sistema actualiza la cita y dispara tu RF-05 (alerta al profesional). (Esto cierra el loop.)
RF-08 — Calificación de asistencia: calcular el % de asistencia / comportamiento histórico del paciente. (Tu diferenciador de Fase 0.)
